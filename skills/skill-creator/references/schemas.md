# Skill Creator Schemas

The following JSON structures are used by the evaluation scripts and subagents to track skill performance.

## 1. `evals.json`
Located in the skill's root or `evals/` directory. Defines the test set.

```json
{
  "skill_name": "example-skill",
  "evals": [
    {
      "id": 1,
      "prompt": "Task for the agent",
      "expected_output": "Description of success",
      "files": []
    }
  ]
}
```

## 2. `timing.json`
Generated by the agent after each subagent run.

```json
{
  "total_tokens": 12345,
  "duration_ms": 5000,
  "total_duration_seconds": 5.0
}
```

## 3. `grading.json`
Generated by the grader subagent after comparing output to assertions.

```json
{
  "eval_id": 1,
  "expectations": [
    {
      "text": "The output contains a valid CSS block",
      "passed": true,
      "evidence": "Observed @theme block on line 12"
    }
  ]
}
```

## 4. `benchmark.json`
Aggregated by `aggregate_benchmark.py` for reporting.

```json
{
  "timestamp": "2024-04-15T12:00:00Z",
  "summary": {
    "pass_rate": 85.0,
    "avg_duration": 4500
  },
  "evals": [
    {
      "id": "eval-1",
      "score": "3/3",
      "passed": true,
      "duration_ms": 4200
    }
  ]
}
```
