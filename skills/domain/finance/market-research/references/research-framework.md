# Investment Research Framework

Use this framework to synthesize high-quality financial research.

## Phase 1: Data Gathering
- **Asset Details**: Name, Ticker, Category, Asset Class.
- **Performance**: 1Y, 3Y, 5Y Returns (Annualized).
- **Risk Metrics**: Standard Deviation, Sharpe Ratio, Maximum Drawdown.
- **Costs**: Expense Ratio, Front-end Load.

## Phase 2: Benchmarking
- **Identify Benchmark**: Select the appropriate index (e.g., SET50, S&P 500, MSCI World).
- **Calculation**: Asset Return - Benchmark Return = **Alpha**.
- **Analysis**: Is the asset consistently beating the benchmark?

## Phase 3: Synthesis (The Report)
1. **Executive Summary**: 2-3 sentences on the core findings.
2. **Performance Review**: Growth vs Benchmark analysis.
3. **Risk Analysis**: Volatility vs Return tradeoff.
4. **Conclusion**: Buy/Hold/Sell/Watch recommendation with reasoning.
